# Sécurité

Merci de ne pas publier publiquement les détails d’une vulnérabilité exploitable.
Contactez d’abord le mainteneur du dépôt GitHub de manière privée.

Ne joignez jamais à un signalement :

- une base `database/app.sqlite` réelle ;
- un fichier `database/database.config.php` ;
- des mots de passe, jetons Mastodon ou fédération ;
- des clés ActivityPub ;
- des adresses électroniques ou pièces jointes d’utilisateurs.

Les recommandations de déploiement sont détaillées dans
[SECURITY_NOTES.md](SECURITY_NOTES.md).
